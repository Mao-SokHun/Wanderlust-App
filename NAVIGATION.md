# Wanderlust — who can do what

| Feature | Guest (no account) | User (logged in) |
|---------|--------------------|------------------|
| Open app → Home | Yes | Yes |
| Search / find places (nearby) | Yes | Yes |
| Browse Tours marketplace | Yes | Yes |
| Open tour / place detail | Yes | Yes |
| **Saved** list | No → register / login | Yes |
| Tap **Save place** | Alert → register / login | Saves to list |
| Business / Admin | No | Role-based |

Startup: Splash → Main (Home). No login wall.
